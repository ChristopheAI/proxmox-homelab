# Proxmox Homelab

Systems-focused homelab documentation and operations repository.

## Context

This homelab has been actively built since **2025** as a practical engineering environment for:
- Linux operations
- Service reliability
- Monitoring and troubleshooting
- Backup and restore discipline

## Core Stack

- Proxmox VE
- Linux containers (LXC)
- DNS and network tooling
- Monitoring and uptime checks
- Backup workflows
- Reverse proxy and secure remote access

## Repository Structure

- `docs/architecture.md`: topology and service layout
- `docs/services.md`: current service inventory
- `docs/networking.md`: network and segmentation notes
- `docs/monitoring.md`: observability setup and alerting principles
- `docs/backup-restore.md`: backup and recovery process
- `runbooks/`: operational procedures for recurring tasks

## Why This Matters

This repository demonstrates production-minded engineering habits:
- reproducible setup notes
- clear operational runbooks
- incident-oriented thinking
- documentation that supports maintenance over time

## Status

Active and continuously documented.
